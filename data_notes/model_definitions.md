
Factor Metrics

Relative Performance
= average sector 1Y return - benchmark 1Y return

Valuation Premium
= average sector Forward P/E / benchmark Forward P/E

Momentum Breadth
= number of sector stocks with positive 1Y return / total valid stocks

Dispersion
= standard deviation of sector 1Y returns

Raw Score = normalized -1 to +1
Display Score = ((Raw Score + 1) / 2) * 100

Cycle Score
= weighted average of factor display scores

Speculation Pressure = normalized bubble/crowding pressure
0 = low pressure
50 = moderate pressure
100 = high pressure

Macro Dashboard Metrics

Cycle Maturity
= average Cycle Score across sectors

AI Divergence 

= cycle score - speculative pressure
-100 <> +100 scale - normalized, asks how far away from the economy at large is the AI economy drifting


Reality Gap
= normalized investor sentiment - normalized consumer sentiment

Liquidity Gap
= AI maturity score - normalized Fed Funds liquidity score

Adoption Gap
= AI maturity score - normalized industrial production score
